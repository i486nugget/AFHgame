# AWE's False Heaven
"AWE's False Heaven" (AFH) is a fangame based off of the tower-based obstacle course game on ROBLOX known as Eternal Towers of Hell (EToH).

- Click [here](https://www.roblox.com/games/12489090264/) to play the game in its latest version.
- Some songs are credited to me in-game, [here's](https://github.com/i486nugget/song-repo) a repository with all of my songs.
